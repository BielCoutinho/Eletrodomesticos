Informações da Aplicação

Este sistema foi criado para facilitar o controle de clientes e ordens de serviço em assistências técnicas de eletrodomésticos.

Resumo do Projeto

Versão: 1.0

Licença: MIT

Desenvolvedor: Gabriel Coutinho

Descrição: A proposta deste projeto é oferecer uma ferramenta prática para organizar dados de clientes e gerenciar ordens de serviço de maneira eficiente.

Principais Recursos

  • Registro completo de clientes com nome, endereço e informações de contato.
  • Lançamento e acompanhamento de ordens de serviço para consertos.
  • Tela intuitiva com design adaptável para diferentes dispositivos.
  • Funcionamento local, sem necessidade de internet.
